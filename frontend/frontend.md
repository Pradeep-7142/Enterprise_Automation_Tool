# Enterprise Workflow Automation Platform – Complete Frontend UI/UX Design Prompt

Build a modern, enterprise-grade React frontend using JavaScript (not TypeScript), HTML, CSS, React Router, and Material UI (MUI v5). The application should look like a premium SaaS product comparable to ServiceNow, Jira, Linear, Notion, Slack, Atlassian Admin, and Monday.com.

The UI must be responsive, clean, elegant, minimal, and highly polished. Every page should feel production-ready rather than like a student project.

---

## Overall Theme

Create a premium enterprise dashboard.

Design language:

* Clean white and light gray surfaces
* Deep blue primary color
* Rounded corners (12–16px)
* Soft shadows
* Glassmorphism only where appropriate
* Smooth animations
* Beautiful hover effects
* Consistent spacing
* Modern typography
* Plenty of whitespace
* Professional icons
* Skeleton loaders
* Empty states
* Loading states
* Success and error animations
* Dark mode support
* Fully responsive

Use Material UI components wherever possible while customizing them to look premium.
